# SymmetryArt
Draw symmetric art and have fun !

## Project for GUI
  * Download / Clone [jar file](https://github.com/saurav-singh/SymmetrySketch/blob/master/Symmetry%20Sketch.jar)
  * Install java
  * In terminal / cmd : java -jar "Symmetry Sketch.jar"
